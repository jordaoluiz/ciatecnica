# Cia Técnica

## Instalação

Para a instalação utilizar o comando:

```bash
npm install
```
dentro da pasta do projeto rode o comando:
```bash
npm start
```

## Técnologias Utilizadas

```
 -> REACTJS
 -> LOCAL STORAGE **PARA ARMAZENAR OS DADOS**
 -> MATERIAL UI
 -> STYLED COMPONENTS
 -> AXIOS
 -> UNFORM
```
